Alien Escape

2-player online co-op escape game made in Unity.

Game link: https://borkens.itch.io/alien-escape The password to access the page is "alien".
Source code: https://github.com/mmarckwo/2p-Escape-Unity

Developed using Unity, Photon Fusion, and Blender.

Alien Escape is a game where two players must work together to escape a facility that they've been captured into. It's a puzzle game with some elements of physics. Players must navigate the rooms and use and share items that they find along the way by throwing them to each other.
Such items include keys, a flashlight, an umbrella, a hammer, and a teleporter.

The game is networked via Photon Fusion. I had to teach myself how to do multiplayer networking with Fusion from scratch over the course of 6 months, all while developing an interesting game behind it. All assets are custom-made by me, with the exception of sounds and music.
