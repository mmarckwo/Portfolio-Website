Global Digital Services - Gateway Website

A website that leads to other collaboratively-made websites between IUPUI and Haaga-Helia students.

Website: https://whelling.pages.iu.edu/globaldigitalservices/
HTML source code: https://github.com/IUPUI-HH/globaldigitalservices/blob/main/2022/index.html
CSS source code: https://github.com/IUPUI-HH/globaldigitalservices/blob/main/2022/styles/style.css

This website was designed and developed by me along with some work between another IUPUI student and a web professor from Haaga-Helia, a university in Helsinki, Finland.

The website exemplifies what can be learned and accomplished by working with an international team. When designing the website, an emphasis was put on accessibility. The page was extensively tested with the W3C Markup Validator to ensure that it satisfies accessibility standards. The HTML for the page fully utilizes semantic tags, and the design and colors for it were carefully developed to maintain readability.
